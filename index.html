Context üè≠

The current state of industrial maintenance and the gaps NextbaseAI addresses.

üìû
Slow Incident Response
When a machine faults, the technician needs to reach an expert. That expert may be unavailable. Diagnosis time depends on who picks up the phone ‚Äî not on the data the machine already has.
üìã
Inaccessible Documentation
Manuals and schematics exist but are rarely consulted on the floor. Finding the right page in a 600-page PDF is slower than trial and error. The information is there ‚Äî the interface to it is missing.
üß†
Tribal Knowledge
Senior technicians carry decades of undocumented knowledge. When they leave, that expertise is lost. Onboarding new staff takes years of accumulated experience that was never written down.
üìä
Underutilized PLC Data
PLCs continuously collect process data (temperatures, pressures, speeds, fault codes). In most factories, this data is only checked after a failure. There's no continuous, intelligent layer between the PLC and the maintenance team.

Approach ‚ö°

NextbaseAI sits between the machine and the maintenance team. It reads PLC data, understands schematics, and communicates via the channels technicians already use.

üí¨
WhatsApp Interface
The primary user interface is WhatsApp ‚Äî no custom app required. Technicians can send text or voice messages. The agent processes natural language queries and responds with structured answers, annotated schematics, and live data.
‚ö°
Real-Time PLC Access
The AnlageAI Gateway runs on-premise and exposes PLC tag values via REST API. The agent can read any configured tag on demand ‚Äî temperatures, pressures, fault codes, motor states ‚Äî without direct PLC access. Read-only by design.
üîå
Schematic Intelligence
Electrical schematics are converted to ESCL (a structured text format we developed). The agent can trace power paths, find cross-references, analyze failure impact, and generate annotated schematic images ‚Äî all programmatically.
üìä
Token-Efficient Monitoring
A separate daemon process polls PLC values at configurable intervals. Alert rules are evaluated deterministically ‚Äî no LLM involved. The AI agent is only woken when a rule triggers, keeping operational cost near zero during normal conditions.
üì±
Mobile PWA
A progressive web app provides a richer chat interface for users who prefer it over WhatsApp. Supports image attachments, audio messages, and a responsive layout. No app store deployment required.
üîí
Isolated Deployment
Each customer runs in a dedicated Docker container. Machine data, conversation history, schematics, and configs are fully isolated. No shared database, no multi-tenant data mixing.
